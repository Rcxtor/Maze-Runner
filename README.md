# Maze Runner

A first-person survival horror maze game built in Unity (URP).

The player spawns inside a dark, sprawling maze inspired by the backrooms 
aesthetic and must navigate to the exit using only a flashlight. The 
flashlight runs on a battery that depletes over time — find battery pickups 
scattered across the maze to survive. When the battery dies, visibility drops 
to zero.

## Built With
- Unity (URP 3D)
- ProBuilder (maze construction)
- C# (game logic & systems)

## Core Features
- First-person flashlight mechanic with real-time battery drain
- Low battery flicker effect for tension
- Battery pickup system for resource management
- Complete darkness on battery death
